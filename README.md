# Craftlanee

This project is migrated to Next.js with Tailwind CSS.

## Setup

1. Install dependencies

```bash
pnpm install
```

2. Run development server

```bash
pnpm dev
```

3. Build for production

```bash
pnpm build
```

## Notes

- Uses Next.js App Router
- Uses Tailwind CSS for responsive styling
- Content is stored in `content.json`
- Theme tokens are stored in `theme.json`
- Contact form uses `@emailjs/browser` client-side
- Sitemap is generated from `src/app/sitemap.ts`
